# 2D横版跑酷游戏

这是一个基于HTML5 Canvas和JavaScript开发的简单而有趣的2D横版跑酷游戏。玩家可以控制角色进行跳跃，躲避障碍物，并获得高分。游戏具有现代化的视觉效果、流畅的动画和响应式设计，支持键盘和触摸操作。

## 游戏特性

### 核心玩法
- **角色控制**：使用键盘或触摸屏幕控制角色跳跃
- **四段跳跃**：角色可以进行连续四次跳跃，增强游戏可玩性
- **障碍物系统**：动态生成障碍物，与前一个障碍物保持最小间隔
- **分数系统**：游戏过程中不断累积分数
- **难度递增**：随着游戏进行，难度会逐渐增加

### 视觉效果
- **角色动画**：角色奔跑动画效果
- **背景元素**：移动的云朵背景，增加游戏深度感
- **响应式设计**：适配不同屏幕尺寸
- **游戏界面**：简洁明了的开始界面和游戏结束界面

### 操作体验
- **优化的跳跃机制**：调整了跳跃高度、滞空时间和移动速度
- **多平台支持**：支持电脑键盘操作和移动设备触摸操作

## 操作说明

### 键盘操作
- **空格键**：跳跃（最多可连续跳跃四次）
- **开始界面**：按空格键开始游戏
- **游戏结束**：按空格键重新开始

### 触摸操作
- **点击屏幕**：跳跃（最多可连续跳跃四次）
- **开始界面**：点击屏幕开始游戏
- **游戏结束**：点击屏幕重新开始

## 运行方式

### 本地运行
1. 确保您的计算机上已安装现代浏览器（推荐Chrome、Firefox、Safari或Edge）
2. 打开终端或命令提示符
3. 导航到游戏目录
4. 启动本地HTTP服务器：
   ```
   python -m http.server 8000
   ```
   或者使用其他HTTP服务器工具
5. 在浏览器中访问 `http://localhost:8000` 开始游戏

### 直接打开
您也可以直接双击 `index.html` 文件在浏览器中打开游戏。

## 游戏参数设置

游戏中的一些核心参数已在 `game.js` 文件中优化设置：

- **跳跃高度**：角色跳跃力设置为-15
- **滞空时间**：角色重力设置为0.4
- **移动速度**：游戏速度设置为3
- **障碍物间隔**：障碍物最小间隔设置为300像素
- **跳跃次数**：角色可连续跳跃4次

## 开发说明

### 技术栈
- **HTML5**：提供游戏的基本结构
- **CSS3**：负责游戏的样式和响应式设计
- **JavaScript**：实现游戏的核心逻辑和交互
- **Canvas API**：负责游戏的渲染和动画效果

### 如何修改游戏

您可以通过编辑 `game.js` 文件来自定义游戏参数：

1. **调整难度**：修改 `gameSpeed` 变量（值越大，速度越快）
2. **改变跳跃特性**：修改 `jumpForce` 和 `gravity` 变量
3. **调整障碍物生成**：修改障碍物生成概率和最小间隔
4. **修改跳跃次数**：调整 `jumpsRemaining` 变量的值
